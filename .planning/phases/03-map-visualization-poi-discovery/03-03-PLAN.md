1. Create `frontend/src/services/pois.api.js`:
   - Import api from './api'
   - Export `async function getPOIs(stopId)`: GET `/stops/${stopId}/pois`, return response.data
   - Export `async function searchPOIs(stopId, query)`: GET `/stops/${stopId}/pois/search?q=${encodeURIComponent(query)}`, return response.data
   - Export `async function getCategories()`: GET `/pois/categories`, return response.data

2. Create `frontend/src/hooks/usePOIs.js`:
   - Import { useState, useEffect, useCallback } from 'react'
   - Import { getPOIs, searchPOIs, getCategories } from '../services/pois.api'
   - Export `function usePOIs(stopId)`:
     - State: `pois` ([]), `categories` ([]), `isLoading` (false), `error` (null), `searchQuery` (null)
     - useEffect on mount: call getCategories() and set categories state (fire-and-forget, no loading state for this)
     - useEffect on stopId change:
       - If no stopId, set pois to [] and return
       - Set isLoading true, error null, searchQuery null
       - Call getPOIs(stopId)
       - On success: set pois to response.pois
       - On error: set error to err.response?.data?.error || 'Failed to load POIs'
       - Finally: set isLoading false
     - Export `search` callback (useCallback):
       - Accepts `query` string
       - If query is empty/null, re-fetch default POIs via getPOIs(stopId) and set searchQuery null
       - Otherwise: set isLoading true, call searchPOIs(stopId, query), set pois and searchQuery
       - Handle errors same as above
     - Return { pois, categories, isLoading, error, searchQuery, search }
